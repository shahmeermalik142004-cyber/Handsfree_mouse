# Handsfree_mouse
Hands-Free Mouse is an open-source, intent-driven cursor control system that allows users to fully operate their mouse using only facial movements.
Instead of direct face-to-cursor mapping, it uses a joystick-style control model with a central deadzone to reduce strain and improve precision. Mouse actions such as left click, right click, drag, and double click are triggered through carefully designed facial gestures like blinks, winks, and jaw movement.

The system focuses on distinguishing intentional actions from natural facial behavior, minimizing false inputs and making hands-free cursor control more reliable and usable.
